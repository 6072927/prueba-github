Comandos LINUX

¿Que es un Comando?

Es un programa o utilidad que se ejecuta en la CLI, es
una forma de interactuar con un programa de computadora o
sistema operativo mediante la introducción de instrucciones, 
textos y procesos. Puedes ejecutar comandos para realizar  
diversas tareas, desde la instalación de paquetes hasta la 
gestión de usuarios y la manipulación de archivos.

Ejemplo de un comando

1. Comandos de navegación

pwd # Muestra la ruta actual
ls # Lista los archivos en el directorio
cd nombre_carpeta # Cambia de directorio

2. Comandos de archivos

touch archivo.txt # Crea un archivo vacío
rm archivo.txt # Borra un archivo
mv origen destino # Mueve o renombra un archivo
cp origen destino # Copia un archivo

3. Permisos y usuarios

chmod 755 archivo # Cambia permisos
chown usuario archivo # Cambia dueño
sudo comando # Ejecuta como administrador

4. Administración de procesos

ps aux # Muestra procesos activos
kill PID # Mata un proceso por ID
top # Muestra consumo de recursos

5. Administración de paquetes

sudo apt update # Actualiza lista de paquetes
sudo apt upgrade # Actualiza los paquetes
sudo apt install paquete # Instala un paquete
sudo apt remove paquete # Desinstala un paquete

6. Comandos de red

ping google.com # Verifica conexión
ifconfig # Muestra configuración de red
netstat -tulnp # Muestra puertos abiertos
